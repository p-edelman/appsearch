About

This app is meant to launch other apps by typing (part of) their name. It is similar to what the Google universal search widget provides (but limited to apps), but much faster.

This app is written as an "itch to scratch"; on my PC, I'm used to start applications by Gnome Do, Launchy or similar apps. The Google search bar on Android is also able to launch apps, but often waits for ~10 seconds before providing results. This app provides results right away (or almost).

The project is quite small and basic and I don't intend to add much to it or provide frequent updates. The source code is available under terms of the GNU General Public License version 3 (GPLv3) and available from the project home (hey, everybody needs a home, rigth)?
